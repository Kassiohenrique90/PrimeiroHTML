Falando um pouco do HTML

(Voltar)

O HTML consiste em 3 partes: HTML,HEAD e BODY.

Head é o cabeçalho, são coisas que ele vai carregar antes de exibir para o usuário, O Head ele tem uma tag filha chamada TITLE que é o titulo da página, O BODY é o corpo tudo o que tiver entre o corpo vai aparecer para o usuário.

Falando sobre Tags

(Voltar)

(i) significa que tudo que estiver entre ele vai está como itálico.

(strong) Deixa tudo em negrito.

Se preciso podemos deixar a frase em negrito e en itálico também.

(input) Essa tag não tem uma teg de fechamento, mas sim ela faz um campo para eu digitar, serve para digitar textos, números e etc.

(img) Serve para adicionar uma imagem e pode ajustar o tamanho do a qual preferir.

(hr) ela coloca uma linha e faz como se tivesse um parágrafo.

Atributos das Tags

Os atributos nada mais são do que propriedades que as tags deve ter, existe atributos mais globais pois todas as tags vão ter e existe atributos específicos de cada tag.

Na tag strong eu posso usar o id esse id ele é o identificador desse elemento que posso usar para fazer uma ancoragem, Vamos ter tbm o style= que é mais voltado para o CSS, pois com isso podemos dar vida trocando de cor tamanho de fonte e etc.

Na tag input eu posso determinar o que eu escrevo, se é tipo texto,número ou cor.

Falando sobre Textos

(Voltar)

Os principais textos usados no dia a dia, h1,h2, h3, h4,h5, h6 - São Voltados para Títulos da minha Página, onde o h1 é o maior e o h6 o menor.

Para cada novo texto usamos a tag (p).

Usamos o blockquote para que no meio do texto queremos fazer uma citação, meio que destacar um texto.

Pra sublinhar um texto usamos a tag (u)

Para textos pequenos usamos (small)

Para usar um marcador de texto utilizamos o (mark)

Para colocar algum caracter em cima de onde vc desejar usamos o (sup)

Listas ordenadas e não ordenadas

(Voltar)

Usamos (OL) Que são as listas ordenadas, para ela funcionar ela precisa das tags filhas que é uma tag que está dentro da outra.

Usamos (UL) é uma lista não ordenada que na tela vai aparecer pontos ao invés de números.

Links no HTML

(Voltar)

Usando a tag (a) podemos colocar um link quando para clicar em determinado local do site.

Usamos o blank para abrir o determinado site em outra página.

Usamos o self para abrir o site na mesma página.
